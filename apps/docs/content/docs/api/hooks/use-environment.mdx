---
title: useEnvironment
description: Returns the current environment - `'development'` or `'production'`.
preview: useEnvironment
indexGroup: Environment Hooks
---

This can be used to determine if the agent is running in a development or deployed server environment.

<Callout type="warn">
    Not to be confused with [`useEnv`](/api/hooks/use-env), which returns environment variables.
</Callout>
